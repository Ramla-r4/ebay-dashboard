eBay API → Backend → AI → Dashboard
Google Sheets → Backend → AI → Dashboard
🛒
Step 1 — eBay API
Connects via OAuth 2.0. Pulls live orders, active listings, and fee transactions from the last 90 days automatically.
📊
Step 2 — Google Sheets
Reads your spreadsheet via service account. Pulls revenue, expenses, inventory costs, and orders you track manually.
🤖
Step 3 — AI Analysis
Both datasets are merged and sent for analysis. Returns plain-English insights, net profit calculation, alerts, and a health score.
📱
Step 4 — Dashboard
Results cached and displayed here. Auto-refreshes daily at 7 AM. One-click manual refresh. Accessible from any browser.
